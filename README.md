# 🍱 Food Donation App

## 🚀 Features
- User Authentication
- Food Request System
- Donor Accept Flow

## 🛠 Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB

## 📁 Project Structure
food-frontend/
food-backend/

## ⚙️ Run Locally

### Backend
cd food-backend
npm install
npm start

### Frontend
cd food-frontend
npm install
npm start
